# RETAIL_ANALYTICSDB
A retail database that stores various categories of products, performs a performance check of various products, promotion, trends and so forth.
Project that implements a relational database model for a supermarket designed to analyze sales, performance, trends, product demand and promotions.
DATABASE SUPPORTS:
Trending products detection, price history, sales volume and revenue tracking, data accuracy and consistency validation
ANALYTICS CAPABILITIES
TRENDING PRODUCTS:
Supports daily, weekly and or monthly trend analysis
identifies fast-growing products using sales growth rates
PRICING ANALYSIS
Tracks price changes over time
Enables elasticity and revenue impact analysis
